# Folder File Organizer Automation

## Problem
Many users keep all files in one folder (Downloads/Desktop),
making it difficult to manage files manually.

## Solution
This Python script automatically organizes files into folders
based on file types (PDF, Images, Documents, Videos, etc.).

## Tools Used
- Python
- os / pathlib module

## How to Run
1. Place the script in the target folder
2. Run: python organizer.py
3. Files will be automatically organized

## Use Case
- Office staff
- Freelancers
- Students
- Small businesses
